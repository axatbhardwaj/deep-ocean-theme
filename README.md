# Deep Ocean Theme for Zed

A deep ocean dark theme for the [Zed](https://zed.dev) editor. Inspired by Material Theme's Ocean palette with refined contrast and syntax highlighting.

![Deep Ocean Theme](screenshot.png)

## Install

1. Open Zed
2. Open the command palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
3. Search for **zed: extensions**
4. Search for **Deep Ocean** and click **Install**

## Dev Install

1. Clone this repository
2. Open the command palette in Zed
3. Search for **zed: install dev extension**
4. Select the cloned directory
